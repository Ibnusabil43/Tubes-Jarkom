Tubes Jarkom

Pembuatan Web Server yang dapat menerima request data yang dikirimkan klien dan mengolah data request tersebut dan menampilkannya di browser Client seperti file pdf, jpeg, gif, dan lain sebagainya. Web server juga harus dapat mengirimkan response message kepada client dengan pesan "HTTP/1.1 200 OK" Jika berhasil dan "HTTP/1.1 404 Not Found" Jika file tidak ditemukan
